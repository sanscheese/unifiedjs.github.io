Support

There are a couple of places to get support when using unified that are described in this section. Before participating in unifiedâs communities, please read our Code of Conduct. We expect that all community members adhere to the guidelines within.

Questions can be asked on Spectrum. Take a look at support.md to find out how to help us help you. Itâs important to spend enough time making your question clear yet detailed, because generally speaking, if more time is spent on a quality question, less time is needed to give a good answer.

Issues can be raised on GitHub in one of the repositories under our organizations. See contributing.md to find out how to report problems. This document also explains how improve documentation and contribute to unified.

Security issues can be reported by email and are handled through security advisories on GitHub. See security.md to find out how to submit a report.
